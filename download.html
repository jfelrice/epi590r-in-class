About the libpng distributions

These two files contain the source and supporting files with LF line endings (for Unix/Linux). Both have identical content.

  • libpng-<version>.tar.xz (LZMA-compressed)
  • libpng-<version>.tar.gz (deflate-compressed)

These two files contain the source and supporting files with CRLF line endings (for Windows). Both have identical content.

  • lpng<version>.7z (LZMA-compressed)
  • lpng<version>.zip (deflate-compressed)

Problems with downloading or decompressing these files can be reported to png-mng-implement at lists.sourceforge.net.
